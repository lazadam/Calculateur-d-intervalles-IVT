Œdème Maculaire Diabétique (OMD)

Recommandations :

  • Phase d'induction : 5 injections mensuelles
  • Phase de maintenance : adaptation selon la réponse
  • Intervalle typique : 4-8 semaines

Occlusion Veineuse Rétinienne (OVR)

Recommandations :

  • Phase initiale : 6 injections mensuelles
  • Suivi : adaptation selon l'évolution
  • Intervalle typique : 4-8 semaines

DMLA (Dégénérescence Maculaire Liée à l'Âge)

Recommandations :

  • Phase d'induction : 3 injections mensuelles
  • Phase de maintenance : adaptation selon la réponse
  • Intervalle typique : 4-12 semaines

Néovaisseaux du Myope Fort

Recommandations :

  • Phase initiale : 3 injections mensuelles
  • Suivi : adaptation selon l'évolution
  • Intervalle typique : 4-8 semaines
